# Introduction_to_Computation-B-

本仓库为北京大学《计算概论B》作业整理仓库，包含个人学习资料、备忘录和课程心得总结。通过整理这些文件，可以方便自己复习，也方便其他同学参考学习方法。

---

## 仓库内容

- `cheatsheet.docx`  
  考试备忘录（Cheatsheet），总结了课程中的核心概念、重要公式和关键知识点，便于快速复习。

- `notes.docx`  
  课程详细笔记，包含各章节知识点的整理、例题解析以及个人理解和补充内容。

- `reflection.docx`  
  个人课程心得与总结，记录了学习过程中的收获、遇到的挑战以及对未来学习的计划和思考。

---
